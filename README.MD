# ctext - Terminal-based Text Editor in C

ctext is a simple terminal-based text editor written in C for Linux systems. It provides a minimalistic editing experience suitable for quick text editing tasks directly from the command 

## Installation
Clone the repository
```bash
make
```


### Usage: 

#### ctext `<filename>`

 ### Keys:
    CTRL-S: Save
    CTRL-Q: Quit
    CTRL-F: Find string in file
    (ESC to exit search, arrows to navigate)
